========================================
  COMANDOS DA PASTA SISTEMAS  (cola rápida)
========================================

Fluxo normal de todo dia:
1) arrastar/mexer nos arquivos no Finder
2) rodar UM comando de "salvar" abaixo


----------------------------------------
USAR O CLAUDE CODE  (o assistente que mexe nos arquivos por você)
----------------------------------------
1) Abrir o Terminal
2) Entrar na pasta:
     cd ~/sistemas
3) Chamar o assistente:
     claude
4) Digitar o pedido em português, em linguagem normal. Exemplos:
     - "atualize o CLAUDE.md da farmácia com as novas regras de dosagem"
     - "cria um arquivo .bas em VBA que soma os valores da coluna B"
     - "veja o que está pendente no git e explique o que mudou"
     - "faz o commit dessas alterações"
5) Para sair do Claude Code:
     /exit

  - Pode conversar normalmente, sem precisar saber comandos técnicos.
  - Se pedir para "fazer o commit", ele mesmo roda os comandos de git.
  - Fechar o Terminal também sai do Claude Code, mas o /exit é mais educado.


----------------------------------------
ENTRAR NA PASTA (sempre antes de qualquer comando)
----------------------------------------
cd ~/sistemas

  (o começo da linha muda de "~" para "sistemas" = você entrou)


----------------------------------------
SALVAR E ENVIAR PRO GITHUB  (o comando que serve para tudo)
----------------------------------------
cd ~/sistemas && git add -A && git commit -m "atualizacao $(date '+%Y-%m-%d %H:%M')"; git push

  - Use SEMPRE este. Funciona mesmo se não tiver nada novo pra commitar
    (só avisa "nothing to commit" e continua pro push).
  - Guarda a alteração no computador (commit) E manda pro GitHub (push)
    numa tacada só.
  - Se aparecer erro no "push" (em vez de só um aviso), NÃO insista —
    cola a mensagem de erro pro Claude Code resolver.

  Se preferir escrever você mesmo a descrição do que mudou, troque o
  comando acima por dois passos:
    cd ~/sistemas && git add -A && git commit -m "sua descrição aqui"
    git push


----------------------------------------
VER O QUE ESTÁ PENDENTE  (não muda nada, só mostra)
----------------------------------------
cd ~/sistemas && git status


----------------------------------------
VER TUDO QUE O GIT GUARDA  (não muda nada, só lista)
----------------------------------------
cd ~/sistemas && git ls-files


========================================
  LEMBRETES
========================================
- Aspas: use SEMPRE aspas retas ( " ), nunca curvas ( " ).
  Se o Terminal reclamar de aspas, foi isso.
  (Dá pra desligar em: Ajustes do Sistema > Teclado > Texto >
   "Usar aspas e travessões inteligentes")

- Renomear arquivo pelo Terminal (avisa o Git junto):
  git mv caminho/nome_antigo  caminho/nome_novo

- A gaveta guarda RECEITAS (.bas, .py, .html, .md).
  Planilhas com dados de paciente ficam FORA, de propósito.
