# Labs — Regras de extração

Formato compacto para extrair resultados laboratoriais a partir de imagens.

## Formato

```
- LAB (DD/MM/AAAA): parâmetro · parâmetro · parâmetro ...
```

- Parâmetros abreviados, separados por **ponto médio** (·).
- **Valores alterados em negrito**, segundo os intervalos de referência do próprio documento.

## Abreviações padrão

| Abrev. | Parâmetro |
|---|---|
| Hb | Hemoglobina |
| Ht | Hematócrito |
| VCM | Volume corpuscular médio |
| Leuco | Leucócitos |
| Plaq | Plaquetas |
| Creat | Creatinina |
| TFG | Taxa de filtração glomerular |
| TGO | Transaminase oxalacética (AST) |
| TGP | Transaminase pirúvica (ALT) |
| GGT | Gama-GT |
| BT / BD / BI | Bilirrubina total / direta / indireta |
| PCR | Proteína C-reativa |
| Na | Sódio |
| K | Potássio |
| Neut | Neutrófilos |
| Bast | Bastonetes |
| Linf | Linfócitos |
| Mono | Monócitos |
| Seg | Segmentados |

> Lista extensível — acrescentar novas abreviações conforme surgirem, mantendo o padrão.
