# Receituário — Regras do sistema

Ferramenta HTML offline de receituário, standalone.

## Arquivos desta pasta

- `receituario.html` — ferramenta standalone offline **(colar aqui a versão oficial)**
- `medicamentos.json` — base de referência **(colar aqui)**
- `REGRAS.md` — este arquivo

## Características da ferramenta

- Base de **179 itens** em **39 categorias**.
- **Multi-seleção** de medicamentos.
- **Busca em tempo real**.
- Cabeçalho **"USO CONTÍNUO"** no painel de prescrição.
- Funciona **100% offline** (sem dependência de rede).

## Empacotamento

- Distribuída como kit de Claude Project: HTML + JSON de conhecimento + instruções.
